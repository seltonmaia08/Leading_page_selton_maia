Gridverso: Explorando o Universo da Organização Criativa

Este manual foi concebido para auxiliar no desenvolvimento e planejamento de uma oficina para a turma de Edição Digital de Imagens, 2° semestre do curso de Design Digital. Este manual, tem uma abordagem dividida em duas partes. A primeira parte abordar os principais conceitos e como cada parte do grid é denominada. A segunda parte, aborda uma sistematização dos tipos de grid e exemplos práticos reias.

Jogo de Xadrez 3D: Pedras preciosas

Atividade realizada na dicisplina de Modelagem Tridimensional, onde, teriamos que criar um grupo de objetos do mesmo tema para modelar em 3D. O tema escolhido por mim foi um jogo de xadrez muito valioso, onde cada peça é feita por uma pedra ou metal precioso.

Bravo: Caffè e arte - Projeto conceitual de uma cafeteria

A Bravo, é uma cafeteria com um espaço muito aconchegante e inspirador. Seu lema é: "Mais que um café, uma obra de arte!". Ressaltando a excelência de um bom café feito na hora, com todo o seu processo manual. Este ambiente é ideal para artistas que buscam um bom lugar para criar suas artes e, também, para os amantes de um bom café.
